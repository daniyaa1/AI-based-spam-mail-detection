# Dataset

This project uses the **SMS Spam Collection Dataset** from UCI.

- Dataset page: https://archive.ics.uci.edu/ml/datasets/sms+spam+collection
- Download link: https://archive.ics.uci.edu/ml/machine-learning-databases/00228/smsspamcollection.zip

The training script downloads the dataset automatically, so storing a local copy here is optional.
