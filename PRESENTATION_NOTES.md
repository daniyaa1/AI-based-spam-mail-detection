# Presentation Notes

## 1. Problem Statement

Spam messages waste time, reduce trust, and can expose users to phishing or fraud. This project uses NLP and Machine Learning to automatically classify messages as spam or ham.

## 2. Dataset

- SMS Spam Collection Dataset
- 5,572 total messages
- 747 spam messages
- 4,825 ham messages

## 3. Pipeline

1. Data cleaning
2. Text preprocessing
3. TF-IDF vectorization
4. Model training
5. Model comparison
6. Best model deployment in the web app

## 4. Models Compared

- Naive Bayes
- Logistic Regression
- Support Vector Machine (SVM)

## 5. Final Result

- Best model: SVM
- Best accuracy: 98.83%
